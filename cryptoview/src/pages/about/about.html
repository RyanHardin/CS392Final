About

Background

Crypto view is a hybrid mobile application meant to allow users to access information about the cryptocurrencies they are interested on one platform across all of their devices.

API's Used

Get top exchanges by volume for a currency pair. The number of exchanges you get is the minimum of the limit you set and the total number of exchanges available

https://min-api.cryptocompare.com/data/top/


Get the current price of any cryptocurrency in any other currency that you need. If the crypto does not trade directly into the toSymbol requested, BTC will be used for conversion.

https://min-api.cryptocompare.com/data/price


Get open, high, low, close, volumefrom and volumeto from the daily historical data.The values are based on 00:00 GMT time. It uses BTC conversion if data is not available because the coin is not trading in the specified currency.

https://min-api.cryptocompare.com/data/histoday
